About

Robot-Ed provides state-of-the-art education about robotics and associated disciplines within Science, Technology, Engineering, and Math (STEM). We strongly believe that everyone should have an opportunity to learn, interact, and experience robotics.

Robots

Our lessons involve a diverse type of robotic systems, with varying complexity and programming skills.


Edison is a robust educational robot designed to bring coding to life for students and help teachers deliver meaningful 21st century education. Edison can be programmed by drag-and-dropping blocks, a vertical block-based programming language (Scratch), and Python.


mBot is a STEM coding robot for beginners, that makes teaching and learning robot coding simple and fun. Students need to build this robot from scratch, exposing them to all of its basic components. mBot can be programmed using a graphical programming language based on blocks, Python, and C-like code.


Lego Mindstorms is an invention kit with motors and sensors you can use to build interactive robotic creations. Lego Mindstorms gives students the opportunity to design, build and program their own robots using Legos. Lego Mindstorms can be programmed using a graphical programming language based on blocks developed by Lego, as well as other more advanced programming interfaces, such as LabView, RobotC, and Matlab/Simulink.


ST Drone is a drone kit that is fully reconfigurable and programmable. This drone can be operated from any smart phone using an intuitive interface. Students need to assemble the many components involved in this aerial robotic system, learning in the process the role of each component. The drone is ready to fly as it comes with preprogrammed open source C code, which the students can modify.

Programs

At Robot-Ed we offer educational programs for K-12 students for any skill level. Our educational programs are integrative, providing knowledge about individual electrical, mechanical, and software components to complete robotic systems. The programs are designed to enable learning of key robotic concepts in small groups that enable future independent learning towards a career in robotics.
